Spotify Webapplication

Steps how i made Spotify.

All the details are below or click the link to github!

What did i use?

I used nodemon during development to automatically reload the application, i also used an MVC architecture in ExpressJS.

Database

Storage of data which I made using SQLite3, the database I created through a DB connection and SQL queries from the database to TypeORM.

Userinterface

You can load your music of your choice, then you can also go to your favorites art or like your song, you can create playlists and edit them yourself!

Authentication

I chose to use JSON Web Tokens for authentication besides that I have harnessed the passwords with bcrypt!

Express middleware

For the middleware I used a body parser, cookie parser and validation of data with express validator.

Seeding

When the application is started for the first time, I filled the database with seeding data. users table with at least 50 users with randomly chosen passwords. I also put fake users in it with fakerjs to test the data coming true.

Documentation and testing

The API is documented with Swagger, the code and functions are well documented via commenting. I did the tests with Jest!.
